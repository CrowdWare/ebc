Welcome to Ebook Creator

Easily create ebooks (epub) using markdown language. The books are targetted towards Amazon Kindle.

The Ebook Creator is a very easy-to-use e-book creation tool that runs as a desktop application on Linux.
You can create books in EPUB format and PDF format simply by using markdown syntax to format your book.
For example, if you want to create a title, just add a hashtag # in front of the title text and the title will become automatic formatted as a title and also appear later in the table of contents.
For a subtitle, just use two hashtags ## and the title will become a subtitle in the table of contents.
If you want to display a word in Italic, enter an asterisk before and after the word *Italic*.
If you want to display a word in bold, then enter two asterisks before and after the word **bold**.
